# The Cozy Place — Digital Menu

Premium café & restaurant digital menu built for QR code scanning at tables.

## 🚀 GitHub Pages Deployment

1. Upload all files to your GitHub repository
2. Go to **Settings → Pages → Source**: select `main` branch, `/ (root)`
3. Your site will be live at `https://yourusername.github.io/repo-name/`

## ✨ Features

- **Responsive**: Mobile-first, works on all devices
- **Animated hero**: Parallax background, floating particles, animated rings
- **Premium cards**: Hover elevation, tilt effect, staggered entrance
- **Category filter**: Filter menu by Drinks / Food / Coffee / Cold
- **Google Reviews**: QR code + direct link to review page
- **Instagram**: Social integration
- **Smooth scroll**: All transitions are CSS/JS animated

## 🎨 Design

- Fonts: Cormorant Garamond (serif) + Jost (sans-serif)
- Palette: Warm cream, gold, dark brown
- Style: Luxury minimalist café aesthetic
